# Mini Minecraft (Browser)

A tiny Minecraft-inspired sandbox made with plain HTML/CSS/JavaScript.

## Run

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Controls

- Move: `A/D` or `←/→`
- Jump: `W` / `Space` / `↑`
- Mine block: Click
- Place selected block: `Shift + Click`
- Change selected block: use the inventory buttons
